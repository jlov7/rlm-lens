# Roadmap — RLM-Lens

## v0.1 (demo-ready MVP)
- Index folder + demo corpus
- Run query using RLM
- Citations + evidence viewer
- Live trace viewer
- Export bundle + replay
- CI + tests + docs

## v0.2 (polish + power)
- Run comparison (diff traces)
- Coverage heatmap per file/dir
- Better citation validation + auto-fix
- Search within traces
- Provider abstraction improvements

## v0.3 (team features)
- Multi-corpus workspace
- Shared export viewer (static HTML)
- Optional LAN mode with auth token
- Redaction customization profiles

## v1.0 (production-hardening)
- Role-based access (if hosted)
- Strong sandbox isolation mandatory
- Audit log + immutable trace storage
- Formal threat model review
